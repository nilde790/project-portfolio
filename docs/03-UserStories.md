# User Stories

## Authentication

- As a **visitor**, I want to register with my email and password, so 
  that I can create and manage my own listings
- As a **registered user**, I want to log in with my credentials, so 
  that I can access my account and listings
- As a **registered user**, I want to log out, so that I can secure my 
  account on a shared device
- As a **registered user**, I want to edit my profile information, so 
  that my contact details stay up to date

## Listings management

- As a **registered user**, I want to create a listing with photos, 
  price, and description, so that I can sell an instrument I no longer use
- As a **registered user**, I want to upload up to 5 photos per listing, 
  so that buyers can clearly see the instrument's condition
- As a **registered user**, I want to edit my own listing, so that I can 
  correct or update information after publishing it
- As a **registered user**, I want to delete my own listing, so that I 
  can remove it once I no longer wish to sell it
- As a **registered user**, I want to mark my listing as sold, so that 
  buyers stop contacting me about an unavailable item

## Browsing & search

- As a **visitor**, I want to see a homepage with available listings, so 
  that I can browse without needing an account
- As a **visitor**, I want to filter listings by category (e.g. guitars, 
  keyboards), so that I can quickly find the type of instrument I'm 
  looking for
- As a **visitor**, I want to search listings by keyword, so that I can 
  find a specific instrument or brand
- As a **visitor**, I want to filter listings by price range, so that I 
  can stay within my budget
- As a **visitor**, I want to view a listing's full details, so that I 
  can evaluate the instrument before contacting the seller

## Contact

- As a **visitor**, I want to see the seller's contact information on a 
  listing, so that I can reach out directly to negotiate a purchase

  ## Roles & moderation

- As an **admin**, I want to view all listings regardless of owner, so 
  that I can monitor the platform's content
- As an **admin**, I want to delete any listing (not just my own), so 
  that I can remove inappropriate or spam content
- As a **registered user**, I want my role to default to "user" upon 
  registration, so that only a designated admin has elevated permissions
